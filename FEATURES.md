## Pairing System

Ghost is designed for only two people.

There are:

- No usernames
- No searching users
- No friend requests
- No public profiles
- No chat requests

Instead, users connect using a secure one-time invite code.

Flow

1. User registers.
2. User logs in.
3. User taps "Generate Invite Code".
4. System generates a secure 8-character alphanumeric invite code.

Example

HX7P-9QKL

Rules

- Randomly generated.
- Cryptographically secure.
- Expires after 10 minutes.
- Single use only.
- Invalid after successful pairing.
- Cannot be reused.
- Only one active partner per account.
- Once paired, a permanent private chat is automatically created.
- The invite code is immediately destroyed after pairing.
- Users can unpair only from Settings after confirmation.
